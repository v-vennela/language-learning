Edit Course: Spanish for Beginners

Lesson 1: Getting Started with Spanish

Alphabet & Pronunciation:

Vowels: a, e, i, o, u (ah, eh, ee, oh, oo)
Special letters: ñ, ll, rr

Essential Phrases:

Hola = Hello, Buenos días = Good morning, Buenas tardes = Good afternoon, Buenas noches = Good night, Gracias = Thank you, Por favor = Please, De nada = You're welcome, ¿Cómo estás? = How are you?, Muy bien = Very well, Adiós = Goodbye

Numbers 0-10:

0 = cero, 1 = uno, 2 = dos, 3 = tres, 4 = cuatro, 5 = cinco, 6 = seis, 7 = siete, 8 = ocho, 9 = nueve, 10 = diez

Lesson 2: Basic Grammar

Personal Pronouns:

Yo = I, Tú = You (informal), Él/Ella = He/She, Nosotros = We, Ellos/Ellas = They

Basic Verbs (Present Tense):

Ser = To be (permanent)

Yo soy = I am, Tú eres = You are, Él/Ella es = He/She is

Estar = To be (temporary)

Yo estoy = I am, Tú estás = You are, Él/Ella está = He/She is

Tener = To have

Yo tengo = I have, Tú tienes = You have, Él/Ella tiene = He/She has

Essential Vocabulary:

Days:

Lunes = Monday, Martes = Tuesday, Miércoles = Wednesday, Jueves = Thursday, Viernes = Friday, Sábado = Saturday, Domingo = Sunday

Colors:

Rojo = Red, Azul = Blue, Verde = Green, Amarillo = Yellow, Negro = Black, Blanco = White

Lesson 3: Spanish Sentence Structure

Articles:

Definite (the):

El = masculine, La = feminine

Indefinite (a, an):

Un = masculine, Una = feminine

Word Order:

In Spanish, the typical word order is Subject-Verb-Object (SVO), similar to English.

Questions:

To form questions, the word order changes slightly or a question mark is added. For example: ¿Cómo estás? (How are you?)

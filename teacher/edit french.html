Edit Course: French for Beginners

Lesson 1: First Steps in French

Alphabet & Pronunciation:

Vowels: a, e, i, o, u
Special characters: é, è, ê, ë, ç, â, î, ô, û
Silent letters rule
Liaison concept

Essential Phrases:

Bonjour = Hello, Au revoir = Goodbye, S'il vous plaît = Please, Merci = Thank you, De rien = You're welcome, Comment allez-vous? = How are you?, Très bien = Very well, Excusez-moi = Excuse me, Je ne comprends pas = I don't understand, Parlez lentement = Speak slowly

Numbers 0-10:

0 = zéro, 1 = un, 2 = deux, 3 = trois, 4 = quatre, 5 = cinq, 6 = six, 7 = sept, 8 = huit, 9 = neuf, 10 = dix

Lesson 2: Basic French Grammar

Personal Pronouns:

Je = I, Tu = You (informal), Il/Elle = He/She, Nous = We, Vous = You (formal/plural), Ils/Elles = They

Basic Verbs (Present Tense):

Être = to be

Je suis = I am, Tu es = You are, Il/Elle est = He/She is

Avoir = to have

J'ai = I have, Tu as = You have, Il/Elle a = He/She has

Aller = To go

Je vais = I go, Tu vas = You go, Il/Elle va = He/She goes

Essential Vocabulary:

Days:

Lundi = Monday, Mardi = Tuesday, Mercredi = Wednesday, Jeudi = Thursday, Vendredi = Friday, Samedi = Saturday, Dimanche = Sunday

Colors:

Rouge = Red, Bleu = Blue, Vert = Green, Jaune = Yellow, Noir = Black, Blanc = White

Lesson 3: Practical French

Articles:

Definite:

Le = The (masculine), La = The (feminine), Les = The (plural)

Indefinite:

Un = A (masculine), Une = A (feminine), Des = Some (plural)

Prepositions:

À = At/to, De = From, En = In, Sur = On, Sous = Under, Avec = With, Sans = Without

Common Phrases:

Où est…? = Where is…?, Combien ça coûte? = How much does it cost?, Pouvez-vous m'aider? = Can you help me?, Je voudrais… = I would like…

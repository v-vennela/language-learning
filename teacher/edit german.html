Edit Course: German for Beginners

Lesson 1: Getting Started with German

Alphabet & Pronunciation:

Special characters: ä, ö, ü, ß
Letters different from English: w (v), v (f), j (y)
Common sounds: ch, sch, ei, ie

Essential Phrases:

Hallo = Hello, Tschüss = Bye, Bitte = Please, Danke = Thank you, Guten Morgen = Good morning, Guten Tag = Good day, Guten Abend = Good evening, Wie geht's? = How are you?, Sehr gut = Very good, Sprechen Sie langsam = Speak slowly

Numbers 0-10:

0 = Null, 1 = Eins, 2 = Zwei, 3 = Drei, 4 = Vier, 5 = Fünf, 6 = Sechs, 7 = Sieben, 8 = Acht, 9 = Neun, 10 = Zehn

Lesson 2: Basic Grammar

Personal Pronouns:

ich = I, du = you (informal), er/sie/es = he/she/it, wir = we, ihr = you (plural informal), Sie = you (formal), sie = they

Basic Verbs (Present Tense):

sein = to be

ich bin = I am, du bist = you are, er/sie/es ist = he/she/it is

haben = to have

ich habe = I have, du hast = you have, er/sie/es hat = he/she/it has

Essential Vocabulary:

Days:

Montag = Monday, Dienstag = Tuesday, Mittwoch = Wednesday, Donnerstag = Thursday, Freitag = Friday, Samstag = Saturday, Sonntag = Sunday

Colors:

rot = red, blau = blue, grün = green, gelb = yellow, schwarz = black, weiß = white

Lesson 3: German Structure

Articles:

Definite (the):

der = the (masculine), die = the (feminine), das = the (neuter)

Indefinite (a):

ein = a (masculine), eine = a (feminine), ein = a (neuter)

Word Order:

In German, the verb often comes second in statements and first in questions. Example: Ich gehe nach Hause (I go home). Gehst du nach Hause? (Do you go home?)
